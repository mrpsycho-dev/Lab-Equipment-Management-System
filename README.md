# Lab Equipment Management System (LEMS)

A simple console-based C++ application for managing lab equipment, borrowing records, and technician activities using basic programming concepts.

## Features

- Admin and Lab Technician roles with separate dashboards  
- Add, update, delete, and view lab equipment  
- Search and sort equipment records  
- Borrow and return equipment  
- Track borrowing records and activity history  

---

## How to Run

1. Make sure you have a C++ compiler installed (e.g., g++).

2. Save the code in a file:
   lems.cpp

3. Open terminal / command prompt in the file directory.

4. Compile the program:
   g++ lems.cpp -o lems

5. Run the program:
   lems.exe
